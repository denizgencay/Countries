# CountriesApp
 
An Android application for getting countries from GeoDb.

- Keywords
- [MVVM]
- [Single Activity]
- [Jetpack Navigation]
- [Room Db]
- [Hilt]
- [ViewModel]

- [ViewBinding]
- [LiveData]
- [Kotlin Coroutines]

#Description

Presenting data with instant api response and local database response for usage of both concepts.
